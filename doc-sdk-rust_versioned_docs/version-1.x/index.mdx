# Overview

The SurrealDB SDK for Rust enables simple and advanced querying of a remote or embedded database from client-side or server-side code. Remote connections automatically reconnect when the connection is terminated.

To contribute to this documentation, submit an Issue or Pull Request in the [`docs.surrealdb.com`](https://github.com/surrealdb/docs.surrealdb.com) repository.

To contribute to the SDK code, submit an Issue or Pull Request in the [`surrealdb`](https://github.com/surrealdb/surrealdb) repository.

:::note
___NOTE:___ This SDK is compatible with V1.x.x
:::
