## `<bool>`

The &lt;bool&gt; casting function converts a value into a boolean.

```surql
SELECT * FROM <bool> "true";

true
```

```surql
SELECT * FROM <bool> "some truthy value";

true
```

```surql
SELECT * FROM <bool> 1.375;

true
```
```surql
SELECT * FROM <bool> 1.375;

false
```
