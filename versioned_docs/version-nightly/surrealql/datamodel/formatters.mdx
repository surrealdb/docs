# Formatters

# Formatters

Formatting functions in SurrealQL accept certain text formats for date/time formatting. They're listed down below.

### Date formatters

<table>
<thead>
  <tr>
    <th>Specifier</th>
    <th>Example</th>
    <th>Description</th>
  </tr>
</thead>
<tbody>
  <tr>
    <td>%Y</td>
    <td>2001</td>
    <td>The full proleptic Gregorian year, zero-padded to 4 digits.</td>
  </tr>
  <tr>
    <td>%C</td>
    <td>20</td>
    <td>The proleptic Gregorian year divided by 100, zero-padded to 2 digits.</td>
  </tr>
  <tr>
    <td>%y</td>
    <td>01</td>
    <td>The proleptic Gregorian year modulo 100, zero-padded to 2 digits.</td>
  </tr>
  <tr>
    <td>%m</td>
    <td>07</td>
    <td>Month number (01 to 12), zero-padded to 2 digits.</td>
  </tr>
  <tr>
    <td>%b</td>
    <td>Jul</td>
    <td>Abbreviated month name. Always 3 letters.</td>
  </tr>
  <tr>
    <td>%B</td>
    <td>July</td>
    <td>Full month name.</td>
  </tr>
  <tr>
    <td>%h</td>
    <td>Jul</td>
    <td>Same as %b.</td>
  </tr>
  <tr>
    <td>%d</td>
    <td>08</td>
    <td>Day number (01 to 31), zero-padded to 2 digits.</td>
  </tr>
  <tr>
    <td>%e</td>
    <td>8</td>
    <td>Same as %d but space-padded. Same as %_d.</td>
  </tr>
  <tr>
    <td>%a</td>
    <td>Sun</td>
    <td>Abbreviated weekday name. Always 3 letters.</td>
  </tr>
  <tr>
    <td>%A</td>
    <td>Sunday</td>
    <td>Full weekday name.</td>
  </tr>
  <tr>
    <td>%w</td>
    <td>0</td>
    <td>Day of the week. Sunday = 0, Monday = 1, ..., Saturday = 6.</td>
  </tr>
  <tr>
    <td>%u</td>
    <td>7</td>
    <td>Day of the week. Monday = 1, Tuesday = 2, ..., Sunday = 7. (ISO 8601)</td>
  </tr>
  <tr>
    <td>%U</td>
    <td>28</td>
    <td>Week number starting with Sunday (00 to 53), zero-padded to 2 digits.</td>
  </tr>
  <tr>
    <td>%W</td>
    <td>27</td>
    <td>Same as %U, but week 1 starts with the first Monday in that year instead.</td>
  </tr>
  <tr>
    <td>%G</td>
    <td>2001</td>
    <td>Same as %Y but uses the year number in ISO 8601 week date.</td>
  </tr>
  <tr>
    <td>%g</td>
    <td>01</td>
    <td>Same as %y but uses the year number in ISO 8601 week date.</td>
  </tr>
  <tr>
    <td>%V</td>
    <td>27</td>
    <td>Same as %U but uses the week number in ISO 8601 week date (01 to 53).</td>
  </tr>
  <tr>
    <td>%j</td>
    <td>189</td>
    <td>Day of the year (001 to 366), zero-padded to 3 digits.</td>
  </tr>
  <tr>
    <td>%D</td>
    <td>07/08/01</td>
    <td>Month-day-year format. Same as %m/%d/%y.</td>
  </tr>
  <tr>
    <td>%x</td>
    <td>07/08/01</td>
    <td>Locale's date representation.</td>
  </tr>
  <tr>
    <td>%F</td>
    <td>2001-07-08</td>
    <td>Year-month-day format (ISO 8601). Same as %Y-%m-%d.</td>
  </tr>
  <tr>
    <td>%v</td>
    <td>8-Jul-2001</td>
    <td>Day-month-year format. Same as %e-%b-%Y.</td>
  </tr>
</tbody>
</table>

### Time formatters

<table>
<thead>
  <tr>
    <th>Specifier</th>
    <th>Example</th>
    <th>Description</th>
  </tr>
</thead>
<tbody>
  <tr>
    <td>%H</td>
    <td>00</td>
    <td>Hour number (00 to 23), zero-padded to 2 digits.</td>
  </tr>
  <tr>
    <td>%k</td>
    <td>0</td>
    <td>Same as %H but space-padded. Same as %_H.</td>
  </tr>
  <tr>
    <td>%I</td>
    <td>12</td>
    <td>Hour number in 12-hour clocks (01 to 12), zero-padded to 2 digits.</td>
  </tr>
  <tr>
    <td>%l</td>
    <td>12</td>
    <td>Same as %I but space-padded. Same as %_I.</td>
  </tr>
  <tr>
    <td>%P</td>
    <td>am</td>
    <td>am or pm in 12-hour clocks.</td>
  </tr>
  <tr>
    <td>%p</td>
    <td>AM</td>
    <td>AM or PM in 12-hour clocks.</td>
  </tr>
  <tr>
    <td>%M</td>
    <td>34</td>
    <td>Minute number (00 to 59), zero-padded to 2 digits.</td>
  </tr>
  <tr>
    <td>%S</td>
    <td>60</td>
    <td>Second number (00 to 60), zero-padded to 2 digits.</td>
  </tr>
  <tr>
    <td>%f</td>
    <td>026490000</td>
    <td>The fractional seconds (in nanoseconds) since last whole second.</td>
  </tr>
  <tr>
    <td>%.f</td>
    <td>.026490</td>
    <td>Similar to %f but left-aligned.</td>
  </tr>
  <tr>
    <td>%.3f</td>
    <td>.026</td>
    <td>Similar to .%f but left-aligned but fixed to a length of 3.</td>
  </tr>
  <tr>
    <td>%.6f</td>
    <td>.026490</td>
    <td>Similar to .%f but left-aligned but fixed to a length of 6.</td>
  </tr>
  <tr>
    <td>%.9f</td>
    <td>.026490000</td>
    <td>Similar to .%f but left-aligned but fixed to a length of 9.</td>
  </tr>
  <tr>
    <td>%3f</td>
    <td>026</td>
    <td>Similar to %.3f but without the leading dot.</td>
  </tr>
  <tr>
    <td>%6f</td>
    <td>026490</td>
    <td>Similar to %.6f but without the leading dot.</td>
  </tr>
  <tr>
    <td>%9f</td>
    <td>026490000</td>
    <td>Similar to %.9f but without the leading dot.</td>
  </tr>
  <tr>
    <td>%R</td>
    <td>00:34</td>
    <td>Hour-minute format. Same as %H:%M.</td>
  </tr>
  <tr>
    <td>%T</td>
    <td>00:34:59</td>
    <td>Hour-minute-second format. Same as %H:%M:%S.</td>
  </tr>
  <tr>
    <td>%X</td>
    <td>00:34:59</td>
    <td>Locale's time representation.</td>
  </tr>
  <tr>
    <td>%r</td>
    <td>12:34:59 AM</td>
    <td>Hour-minute-second format in 12-hour clocks. Same as %I:%M:%S %p.</td>
  </tr>
  <tr>
    <td>%x</td>
    <td>07/08/01</td>
    <td>Locale's date representation.</td>
  </tr>
  <tr>
    <td>%F</td>
    <td>2001-07-08</td>
    <td>Year-month-day format (ISO 8601). Same as %Y-%m-%d.</td>
  </tr>
  <tr>
    <td>%v</td>
    <td>8-Jul-2001</td>
    <td>Day-month-year format. Same as %e-%b-%Y.</td>
  </tr>
</tbody>
</table>

### Timezones formatters

<table>
<thead>
  <tr>
    <th>Specifier</th>
    <th>Example</th>
    <th>Description</th>
  </tr>
</thead>
<tbody>
  <tr>
    <td>%Z</td>
    <td>ACST</td>
    <td>Local time zone name.</td>
  </tr>
  <tr>
    <td>%z</td>
    <td>+0930</td>
    <td>Offset from the local time to UTC (with UTC being +0000).</td>
  </tr>
  <tr>
    <td>%:z</td>
    <td>+09:30</td>
    <td>Same as %z but with a colon.</td>
  </tr>
</tbody>
</table>

### Date & time formatters

<table>

  </tr>

  </tr>

</table>
