# Text protocol

## WebSocket (text protocol)

The WebSocket protocol allows for easy bi-directional communication with SurrealDB. This allows you to maintain a single connection to run all your queries, but also opens up the possibility of Live Queries!

:::note
Live queries are still under development, though available soon. 
:::

<h3> Live notification </h3>

<p> For every creation, update or deletion on the specified table, a live notification will be sent. Live notifications do not have an ID attached, but rather include the Live Query's UUID in the result object.</p>

```json
{
    "result": {
        "action": "CREATE",
        "id": "0189d6e3-8eac-703a-9a48-d9faa78b44b9",
        "result": {
            "id": "person:8s0j0bbm3ngrd5c9bx53",

}
```

    <span className='class-name'>merge</span>
    <span className='punctuation'>[ thing, data ]</span>
    <span className='function'>method</span>
</div>

<p> This method merges specified data into either all records in a table or a single record </p>

:::note
<em> NOTE:</em> This function replaces the current document / record data with the specified data. If no replacement data is passed it will simply trigger an update.
:::

<table>
    <thead>
        <tr>
            <th colspan="2">Parameter</th>
            <th>Description</th>
        </tr>
    </thead>  
    <tbody>
        <tr>
            <td colspan="2">
                <code>thing</code>
                <l className='yellow'>REQUIRED</l>
            </td>
            <td>
            The thing (Table or Record ID) to select
            </td>
        </tr>
        <tr>
            <td colspan="2">
                <code>data</code>
                <l className='grey'>OPTIONAL</l>
            </td>
            <td>
            The content of the record
            </td>
        </tr>
    </tbody>
</table>
<h3> Example message </h3>

```json
{
    "id": 1,
    "method": "merge",
    "params": [
        "person",
        {
            "active": true
        }
    ]
}
```
<h3> Repsonse </h3>

```json
{
  "id": 1,
  "result": [
      {
          "active": true,
          "id": "person:8s0j0bbm3ngrd5c9bx53",
          "name": "John Doe"
      },
      {
          "active": true,
          "id": "person:s5fa6qp4p8ey9k5j0m9z",
          "name": "Mary Doe"
      }
  ]
}
```
</div>
