# UUIDs

UUIDs represent UUID v4 and v7 values. They can be obtained via either the:
- [`rand::uuid::*` functions](/docs/surrealdb/surrealql/functions/database/rand#randuuidv4)
- [casted from strings](/docs/surrealdb/surrealql/datamodel/casting#uuid)
- or via [string prefixes](/docs/surrealdb/surrealql/datamodel/strings#uuid)

```surql
rand::uuid::v4();
rand::uuid::v7();
<uuid> "a8f30d8b-db67-47ec-8b38-ef703e05ad1b";
u"a8f30d8b-db67-47ec-8b38-ef703e05ad1b";
```
