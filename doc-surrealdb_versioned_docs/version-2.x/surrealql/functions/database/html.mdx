## `html::encode()` <l className='purple'>Since 1.6.0</l>

The `html::encode()` function encodes text to be used as HTML content.
This can be helpful in certain contexts like generating a valid query string for a URL.

```surql title="API DEFINITION"
html::encode(string) -> string
```
The following example shows this function, and its output, when used in a [`RETURN`](/docs/surrealdb/surrealql/statements/return) statement:

```surql
RETURN html::encode("Hello World");

"Hello%20World"
```

<br />

## `html::decode()` <l className='purple'>Since 1.6.0</l>

The `html::decode()` function decodes text from an encoded HTML component.

```surql title="API DEFINITION"
html::decode(string) -> string
```
The following example shows this function, and its output, when used in a [`RETURN`](/docs/surrealdb/surrealql/statements/return) statement:

```surql
RETURN html::decode("Hello%20World");

"Hello World"
```

<br />

## `html::sanitize()` <l className='purple'>Since 1.6.0</l>

The `html::sanitize()` function sanitizes HTML content to prevent XSS vulnerabilities.
This can be helpful in certain contexts like preventing XSS vulnerabilities coming from user generated content.

```surql title="API DEFINITION"
html::sanitize(string) -> string
```
The following example shows this function, and its output, when used in a [`RETURN`](/docs/surrealdb/surrealql/statements/return) statement:

```surql
RETURN html::sanitize("XSS<script>attack</script>");

"XSS"
```
