# Methods

The SurrealDB PHP SDK offers you methods to interact with the database. You can query and mutate any data you want. The SDK offers you a simple yet powerfull methods to interact with a local or a remote SurrealDB database.

## `.connect()` {#connect}

Connects to a local or remote database endpoint.

```php title="Method Syntax"
$db.connect($url, $options)
```
