## `->connect()` {#connect}

Connects to a local or remote database endpoint.

```php title="Method Syntax"
$db->connect($url, $options)
```

## `->close()` {#close}

Closes the persistent connection to the database.

```php title="Method Syntax"
$db->close()
```

## `->use()` {#use}

Switch to a specific namespace and database. 
If only the namespace or database property is specified, 
the current connection details will be used to fill the other property.

```php title="Method Syntax"
$db->use([ "namespace" => $namespace, "database" => $database ])
```
