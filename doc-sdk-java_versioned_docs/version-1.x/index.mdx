# Overview

The SurrealDB SDK for Java enables simple and advanced querying of a remote database from client or server-side code. Remote connections automatically reconnect when the connection is terminated.

To contribute to this documentation, submit an Issue or Pull Request in the [`docs.surrealdb.com`](https://github.com/surrealdb/docs.surrealdb.com) repository.

To contribute to the SDK code, submit an Issue or Pull Request in the [`surrealdb.java`](https://github.com/surrealdb/surrealdb.java) repository.
