# `.connect()`

Establishes a persistent connection to the database.

```python title="Method Syntax"
db.connect()
```

### Example usage
```python
db.connect()

# async
await db.connect()
```

You can close the connection using the [`close()` method](/docs/sdk/python/methods/close).
