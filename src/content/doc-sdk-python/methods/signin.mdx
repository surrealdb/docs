# `.sign_in()` {#signin}

Signs in a user. Returns an authentication token which is saved as part of the current session. You may keep the token
if you wish but you don't have to.

```python title="Method Syntax"
async db.sign_in(username, password)
```

### Example usage
```python
// Authenticate with a root user
const token = db.sign_in('root', 'surrealdbpass');
```

You can invalidate the authentication for the current connection using the [`invalidate()` method](/docs/sdk/javascript/methods/invalidate). Learn more about [handling authentication](/docs/sdk/javascript/core/handling-authentication).
