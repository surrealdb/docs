# `.RawQuery()` {#query}

Runs a set of SurrealQL statements against the database, based on a raw SurrealQL query.

```go title="Method Syntax"
db.RawQuery(sql string, vars map[string]interface{})
```

### Example usage

```go
// Assign the variable on the connection
vars := map[string]interface{}{"table": "person"}
result, err := db.RawQuery("CREATE person; SELECT * FROM type::table($table);", vars)
```
