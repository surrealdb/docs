# `.Insert[T](table, data)` {#insert}

Inserts one or multiple records in the database.

```go title="Method Syntax"
db.Insert[T](table, data)
```

### Example usage

```go
// Insert an entry
person2, err := surrealdb.Insert[Person](db, models.Table("persons"), map[interface{}]interface{}{
    "Name":     "Jane",
    "Surname":  "Smith",
    "Location": models.NewGeometryPoint(-0.12, 22.01),
})
if err != nil {
    panic(err)
}
fmt.Printf("Inserted person with a map: %+v\n", person2)

```
