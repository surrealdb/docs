# Overview

The [SurrealDB SDK for .NET](https://nuget.org/packages/surrealdb.net) enables simple and advanced querying of a remote database from client or server-side code. Remote connections automatically reconnect when the connection is terminated.

To contribute to this documentation, submit an Issue or Pull Request in the [`docs.surrealdb.com`](https://github.com/surrealdb/docs.surrealdb.com) repository.

To contribute to the SDK code, submit an Issue or Pull Request in the [`surrealdb.net`](https://github.com/surrealdb/surrealdb.net) repository.

> [!NOTE]
> The SDK works seamlessly with SurrealDB versions `v2.0.0` to <Version />, ensuring compatibility with the database version.

> [!WARNING]
> This API is not yet fully stabilized and may be subject to change until the SDK reaches `1.0.0`.
