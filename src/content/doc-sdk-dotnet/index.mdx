# Overview

The SurrealDB SDK for .NET enables simple and advanced querying of a remote database from client or server-side code. Remote connections automatically reconnect when the connection is terminated.

To contribute to this documentation, submit an Issue or Pull Request in the [`docs.surrealdb.com`](https://github.com/surrealdb/docs.surrealdb.com) repository.

To contribute to the SDK code, submit an Issue or Pull Request in the [`surrealdb.net`](https://github.com/surrealdb/surrealdb.net) repository.


> [!IMPORTANT]
>  This SDK is not yet compatible with `v2.x` of SurrealDB. If you want to use the SDK you must be on a version`>=1.0.0` but `<2.0`
