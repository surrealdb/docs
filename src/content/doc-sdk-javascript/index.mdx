# Overview

The [SurrealDB SDK for JavaScript](https://npmjs.com/package/surrealdb) is the primary SDK for interacting with SurrealDB over the web. It is designed to be used in a number of environments, including browser-based applications, server-side applications, and serverless functions. The SDK is designed to be lightweight and easy to use, and provides a number of built-in functions for connecting to a database, performing queries, and managing data.


> [!IMPORTANT]
> The SDK works seamlessly with SurrealDB version <Version />, ensuring compatibility with the latest features and improvements of the database system.
