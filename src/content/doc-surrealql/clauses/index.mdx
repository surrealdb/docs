# Clauses

In SurrealQL, clauses are used to configure and query a database. They are used in the following ways:

- [`FROM`](/docs/surrealql/clauses/from): specify the table or view to query
- [`WHERE`](/docs/surrealql/clauses/where): specify a condition that acts as a filter
- [`WITH`](/docs/surrealql/clauses/with): replace the default table iterator with an index iterator.
