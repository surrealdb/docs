# pnpm

Before you can use this SDK for JavaScript regardless of your environment, you need to install and import it into your project.

### Install the SDK

First, install the SurrealDB SDK using your favorite package manager. Here we will use pnpm.

```bash
pnpm install --save surrealdb.js
```

### Initialise your project

```bash
pnpm start
```

### Clear your project

```bash
pnpm clear
```

### Build your project to be ready for deployment

```bash
pnpm build
```

### Deploy your project

```bash
pnpm deploy
```
