# yarn

Before you can use this SDK for JavaScript regardless of your environment, you need to install and import it into your project.

### Install the SDK

First, install the SurrealDB SDK using your favorite package manager. Here we will use yarn.

```bash
yarn install --save surrealdb.js
```

### Initialise your project

```bash
yarn start
```

### Clear your project

```bash
yarn clear
```

### Build your project to be ready for deployment

```bash
yarn build
```

### Deploy your project

```bash
yarn deploy
```
