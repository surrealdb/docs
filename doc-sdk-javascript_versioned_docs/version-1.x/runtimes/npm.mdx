# npm

Before you can use this SDK for JavaScript regardless of your environment, you need to install and import it into your project.

### Install the SDK

First, install the SurrealDB SDK using your favorite package manager. Here we will use npm.

```bash
npm install --save surrealdb.js
```

### Initialise your project

```bash
npm start
```

### Clear your project

```bash
npm clear
```

### Build your project to be ready for deployment

```bash
npm build
```

### Deploy your project

```bash
npm deploy
```
