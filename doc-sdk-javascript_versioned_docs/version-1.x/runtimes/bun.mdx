# bun

Before you can use this SDK for JavaScript regardless of your environment, you need to install and import it into your project.

### Install the SDK

First, install the SurrealDB SDK using your favorite package manager. Here we will use bun.

```bash
bun install --save surrealdb.js
```

### Initialise your project

```bash
bun start
```

### Clear your project

```bash
bun clear
```

### Build your project to be ready for deployment

```bash
bun build
```

### Deploy your project

```bash
bun deploy
```
